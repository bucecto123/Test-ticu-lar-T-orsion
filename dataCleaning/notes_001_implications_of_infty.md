In the context of flow-aggregated network traffic, specific continuous features such as Flow Bytes/s and Flow Packets/s are derived via time-division operations. In highly concurrent or localized network topologies, the capture duration of a microscopic flow may resolve to zero microseconds. According to IEEE 754 floating-point standards, dividing a non-zero byte volume by a zero-duration scalar results in an infinite value ($\infty$). While Python and NumPy's native float64 types can successfully store this inf state, permitting these values to enter the continuous matrix $X \in \mathbb{R}^{m \times n}$ fundamentally breaks the axioms of the underlying linear algebra.

To construct the invariant geometric subspace $U$ representing network equilibrium, the system must compute the Singular Value Decomposition (SVD) of the continuous feature matrix $X$, defined as $X = U \Sigma V^T$. Computationally, finding the principal components (the columns of $V$) requires resolving the eigenvectors of the empirical covariance matrix, $C = \frac{1}{m-1} X^T X$.If a single dimension $j$ in matrix $X$ contains an inf value, the sample mean $\mu_j$ for that dimension becomes $\infty$. Consequently, the mean-centering operation $X_{i,j} - \mu_j$ results in $\infty - \infty$, which IEEE 754 arithmetic strictly evaluates as NaN (Not a Number). The dot product operations required to construct $X^T X$ will immediately propagate this NaN across the entire covariance matrix.

Even if the mean-centering step is bypassed and the uncentered matrix is utilized, the presence of $\infty$ causes catastrophic failure in the eigen-decomposition phase. The singular values $\sigma_i$ (found on the diagonal of $\Sigma$) represent the variance captured by each orthogonal basis vector. Mathematically, an infinite value implies that the variance along that dimension is infinite.Iterative algorithms used to compute SVD (such as the Power Iteration method or the QR algorithm) rely on converging sequences. When multiplying by a matrix containing $\infty$, the projection vectors scale exponentially to infinity in a single step. The algorithm fails to converge, the characteristic polynomial $\det(C - \lambda I) = 0$ becomes undefined, and the mathematical engine throws an unrecoverable LinAlgError.

From a geometric perspective (Axler), calculating an orthogonal projection $P_U(v)$ requires defining a bounded subspace with a finite orthonormal basis. An infinite value implies a coordinate point that exists infinitely far away from the origin. It is mathematically impossible to draw a meaningful orthogonal projection from an infinitely distant coordinate onto a finite subspace.From a thermodynamic perspective (Savelyev), an infinite Flow Bytes/s suggests a state of infinite kinetic energy. A system with infinite energy possesses no bounded equilibrium. Therefore, calculating the thermodynamic residual magnitude $||w||$ of an anomaly becomes impossible, as the baseline "normal" state mathematically requires the system to be bound by finite energy constraints. Thus, the rigorous sanitization of non-finite values is not merely a programmatic necessity, but a strict requirement to maintain the topological validity of the Euclidean space.
